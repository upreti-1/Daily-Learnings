# Random Forest

This folder contains practice material for learning and implementing Random Forest models.

## Contents

- `Random Forest Implementation.ipynb`: End-to-end Random Forest workflow for training and evaluation.
- `Travel.csv`: Dataset used in the notebook for model building and experimentation.

## Topics Covered

- Ensemble learning with bagging
- Decision trees inside Random Forest
- Training a Random Forest model with scikit-learn
- Model prediction and performance evaluation
- Feature importance analysis
- Basic hyperparameter tuning concepts

## How To Use

1. Open this folder in VS Code.
2. Launch and run `Random Forest Implementation.ipynb` cell by cell.
3. Ensure `Travel.csv` is in the same folder as the notebook.
4. Install required libraries if needed.

Suggested packages:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

## Learning Goal

Build intuition for how Random Forest improves over a single decision tree and learn how to evaluate and interpret model performance on a real dataset.
