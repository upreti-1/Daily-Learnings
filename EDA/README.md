# EDA - Exploratory Data Analysis

This folder contains notebooks for exploratory data analysis on various datasets including flight prices, Google Play Store apps, and red wine quality.

## Notebooks

- `flight price prediction.ipynb` — Flight price EDA and prediction
- `Google playstore dataset.ipynb` — Play Store app analysis
- `redwinedata.ipynb` — Wine quality analysis
- `2.0-EDA+And+FE+Flight+Price.ipynb` — Flight price EDA with feature engineering

## Datasets

- `winequality-red.csv` — Red wine quality data
- `flight_price.xlsx` — Flight price data
- `data/google_cleaned.csv` — Cleaned Play Store data

## Quick Start

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
jupyter lab
```
