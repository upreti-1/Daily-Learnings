# Flask Framework

This folder contains Flask applications and examples demonstrating web development with Flask.

## Contents

- `app.py` — Basic Flask application setup
- `main.py` — Main Flask application entry point
- `api.py` — REST API endpoints using Flask
- `getpost.py` — Handling GET and POST requests
- `jinga.py` — Jinja2 templating examples
- `templates/` — HTML templates for Flask routes
  - `index.html` — Home page
  - `about.html` — About page
  - `form.html` — Form page
  - `result.html` — Result display page
  - `result1.html` — Alternative result page
  - `getresult.html` — GET request result page

## Quick Start

```bash
pip install flask
python app.py
```

Visit `http://localhost:5000` in your browser.

## Topics Covered

- Basic routing and views
- Request handling (GET/POST)
- Template rendering with Jinja2
- REST API development
