# Logistic Regression

This folder contains practice notebooks for learning and implementing Logistic Regression for classification tasks.

## Contents

- `logistic regression implementation.ipynb`: Core Logistic Regression implementation workflow.
- `logistic reg with ROC curve and ROC AUC score.ipynb`: Model evaluation using ROC Curve and ROC AUC Score.

## Topics Covered

- Binary classification with Logistic Regression
- Model training and prediction
- Confusion matrix and classification metrics
- ROC Curve visualization
- ROC AUC score interpretation

## How To Use

1. Open this folder in VS Code.
2. Run notebook cells in order from top to bottom.
3. Install required Python libraries if needed.

Suggested packages:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

## Learning Goal

Understand how Logistic Regression works for classification and how to evaluate performance using ROC-based metrics beyond basic accuracy.
