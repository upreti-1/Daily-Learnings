# OOPS - Object-Oriented Programming in Python

This folder contains notebooks covering core OOP concepts and principles in Python.

## Topics Covered

- `OOPS.ipynb` — Introduction to OOP fundamentals
- `abstraction.ipynb` — Data abstraction and abstract classes
- `encapsulation.ipynb` — Encapsulation and data hiding
- `Inheritance.ipynb` — Class inheritance and method resolution
- `polymorphism.ipynb` — Polymorphism and method overriding
- `operator overloading.ipynb` — Overloading operators with magic methods
- `customexception.ipynb` — Creating and handling custom exceptions
- `assignments.ipynb` — Practice exercises and assignments
- `magic  Methods/magicmethods.ipynb` — Special methods (__init__, __str__, etc.)

## Quick Start

```bash
jupyter lab
```

Then open any notebook to explore the concepts.
