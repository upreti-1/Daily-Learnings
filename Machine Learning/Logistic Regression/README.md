# Logistic Regression

This folder contains hands-on notebooks for learning and implementing Logistic Regression using scikit-learn.

## Contents

1. `logistic regression implementation.ipynb`
   - End-to-end Logistic Regression workflow.
   - Typical steps include data preparation, model training, prediction, and evaluation.

2. `logistic reg with ROC curve and ROC AUC score.ipynb`
   - Focuses on classification performance analysis.
   - Includes ROC curve plotting and ROC-AUC score interpretation.

## Learning Objectives

- Understand how Logistic Regression works for binary classification.
- Train and evaluate a classifier using standard ML workflow.
- Interpret confusion matrix, ROC curve, and ROC-AUC metrics.

## Requirements

Recommended Python packages:

- numpy
- pandas
- matplotlib
- seaborn
- scikit-learn
- jupyter

Install with:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

## How To Run

1. Open this folder in VS Code or Jupyter.
2. Launch the notebook you want to study.
3. Run cells in order from top to bottom.

## Notes

- Keep dataset file paths updated if running notebooks from a different working directory.
- For reproducible splits/results, use a fixed random seed where applicable.
