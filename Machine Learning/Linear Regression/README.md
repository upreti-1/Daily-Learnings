# Linear Regression

This folder contains practice notebooks and datasets for learning core regression techniques in machine learning.

## Contents

- `Linear regression.ipynb`: Simple Linear Regression workflow.
- `multiple linear regression.ipynb`: Multiple Linear Regression with more than one input feature.
- `polynimial regression.ipynb`: Polynomial Regression for non-linear relationships.
- `height-weight.csv`: Sample dataset typically used for basic/simple regression examples.
- `economic_index.csv`: Dataset for multiple feature regression practice.

## Topics Covered

- Data loading and basic preprocessing
- Train-test split
- Model training with linear models
- Prediction and evaluation
- Visual interpretation of regression behavior

## How To Use

1. Open this folder in VS Code.
2. Run notebooks in sequence from top to bottom.
3. Ensure required Python libraries are installed.

Suggested packages:

```bash
pip install pandas numpy matplotlib scikit-learn jupyter
```

## Learning Goal

Use these notebooks to understand when to apply:

- Simple Linear Regression
- Multiple Linear Regression
- Polynomial Regression

and how model complexity affects fit and generalization.
